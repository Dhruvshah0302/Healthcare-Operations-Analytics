# 🏥 Hospital Administration Dashboard

## 📌 Project Overview
An interactive Power BI dashboard analyzing 19 months 
of hospital emergency room data (April 2023 – October 2024), 
covering 9,216 unique patients across multiple KPIs.

## 🎯 Objectives
- Analyze patient wait times and satisfaction scores
- Identify peak busy periods for staffing optimization
- Track admission patterns and departmental referrals
- Monitor month-over-month performance trends

## 📊 Key Insights
- Average wait time: 35.3 minutes
- Patient satisfaction: 4.99/10
- Busiest day: Monday (1,377 patients)
- Top referral department: General Practice (1,840 cases)
- Nearly equal split: 4,612 admitted vs 4,604 treated & released

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV
